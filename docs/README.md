# Timetable-Maker Documentation

This folder contains comprehensive documentation for the Timetable Generation System.

## Documentation Files

- **PROJECT_AUDIT.md** - Complete project structure and file audit
- **IMPLEMENTATION_PROGRESS.md** - Development progress and feature implementation tracking
- **SETTINGS_PERSISTENCE.md** - Database settings and persistence layer documentation
- **SOLVER_DOCS.md** - Timetable solver algorithm and constraint optimization docs
- **TIME_ALLOCATION_SYSTEM.md** - Time allocation and scheduling system documentation
- **VERSION_MANAGEMENT.md** - Version control and release management

## Quick Links

## Architecture Overview

The system uses:
- **Backend**: Python FastAPI + SQLAlchemy + SQLite
- **Frontend**: React + TypeScript + Vite
- **Future**: PostgreSQL + Drizzle ORM (prepared but not yet active)
